# PodsetnikCSharp — primeri za studente

### Svrha
Ovo rešenje sadrži zbirku malih i fokusiranih primera u C# jeziku, namenjenih studentima koji uče programiranje.

Cilj je da se kroz kratke, jednostavne projekte prikažu najvažnije osobine jezika i tipični obrasci koji se koriste u praksi.

### Status

### Primeri su u .NET 8

Materijal je dodatak za učenje, ne kompletna referenca jezika.

Primeri pokrivaju osnovne i srednje napredne teme, ali ne sve mogućnosti C#.

### Šta se nalazi u projektu
Svaki folder (npr. Primer01, Primer02, ..., Primer10, Primer04-1) sadrži mali konzolni projekat koji prikazuje po jedan koncept.
Uobičajeni fajlovi su Program.cs, Osoba.cs, Student.cs, Kalkulator.cs i slični.
Rešenje se otvara u Visual Studio-u, gde možeš pregledati kod i odmah ga pokrenuti.

### Uputstvo

Kloniraj repozitorijum.
Otvori .sln fajl u Visual Studio 2022.
Postavi željeni projekat kao Startup Project i pokreni ga (opcija Start Debugging ili komanda dotnet run u folderu projekta).

### Doprinosi

Ovaj repozitorijum je otvoren za dopune i predloge.
Ako primetiš grešku, nedostajuću temu ili imaš bolji primer:

- Forkuj repo i napravi novu granu.
- Dodaj ili poboljšaj primer, komentare ili kratke testove.
- Pošalji Pull Request sa kratkim opisom izmene.

#### Smernice

- Drži primere kratkim i fokusiranim.
- Koristi jasna imena i komentare (na srpskom).
- Ako primer koristi dodatne klase (Program.cs, Osoba.cs, itd.), zadrži konzistentnu strukturu.

#### Napomena za studente
- Ovo je pomoćni materijal za razumevanje jezika C#.
- Ne pokriva sve mogućnosti, već ilustruje najvažnije pojmove.
- Za detaljna objašnjenja i kompletnu dokumentaciju koristi Microsoft Learn i docs.microsoft.com.
